procedural


def fact(i):
    if i >= 1:
        return i * fact(i - 1)
    else:
        return 1
Fairly typical recursive definition of factorial.
Things to note:
  • No curly braces!
  • No typing information!
  • "def" introduces a function definition.

functional


def double_fact(i):
    return map(lambda x: fact(x) * 2,
               range(i))
